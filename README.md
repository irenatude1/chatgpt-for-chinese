## ChatGPT 是什么？

ChatGPT 是由 OpenAI 开发的人工智能对话工具，基于大型语言模型技术，能够理解并生成自然语言文本。

你可以把它理解为一个智能助手，能够帮助你完成：

* 写邮件、写文章、写文案
* 翻译和润色文本
* 编写、解释和调试代码
* 总结资料、整理笔记
* 制定学习计划、旅行计划
* 进行多轮中文对话

目前，GPT-4o 是常用的高性能模型之一，具备更快的响应速度，并支持文本、图像、语音等多模态交互体验。

## 新手如何向 ChatGPT 提问？

想获得更高质量的回答，关键在于写好 Prompt。

推荐使用这个公式：

```text
角色 + 任务 + 要求 + 背景 = 高质量回答
```

### 示例

```text
你现在是一名专业旅行规划师。
请为我设计一个为期 5 天的东京自由行计划。
要求包含至少一个动漫主题景点和一个自然风光景点，预算尽量经济实惠。
我是一个人旅行，喜欢美食和摄影。
```

### 提问技巧

* 明确说明你想要什么
* 给出背景信息
* 指定输出格式
* 说明字数、语气或风格
* 不满意时继续追问和修改

## 常见问题 FAQ

### Q1：ChatGPT 是完全免费的吗？

ChatGPT 官方提供免费版本，但部分高级功能通常需要订阅 ChatGPT Plus。国内镜像网站一般会提供免费试用额度，具体以网站实际规则为准。

### Q2：国内 ChatGPT 镜像网站安全吗？

建议优先选择支持 HTTPS、口碑较好、使用人数较多的平台。无论使用官网还是镜像站，都不要输入密码、身份证号、银行卡号、验证码等敏感信息。

### Q3：ChatGPT 中文版和官网数据互通吗？

一般不互通。镜像网站通常通过 API 调用模型服务，对话记录可能保存在镜像网站本身，而不是 OpenAI 官网账户中。

### Q4：GPT-4o 和 GPT-4 有什么区别？

GPT-4o 在响应速度、多语言能力和多模态体验方面表现更好，支持文本、图像和语音等交互方式，适合日常高频使用。

### Q5：国内用户更适合用官网还是镜像站？

如果你追求稳定、官方、完整功能，可以选择官网；如果你更看重访问方便、中文体验和低门槛使用，可以选择国内镜像站。

## 数据隐私与安全建议

使用任何 AI 工具时，都应注意以下几点：

* 不要输入个人敏感信息
* 不要上传重要证件、合同、银行卡截图
* AI 生成内容需要人工核实
* 重要决策不要完全依赖 AI
* 优先选择可信赖的平台
* 定期清理不必要的对话记录
